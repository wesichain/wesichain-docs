Wesichain is a modular Rust workspace.
 Install only the crates you need.

## 🚀 Quick Start with AI Assistance (Recommended)

The fastest way to build with Wesichain is using Claude Code with the **AI Skills Pack**:

1. **Download** the latest `wesichain.skills` from [GitHub Releases](https://github.com/wesichain/wesichain/releases)
2. **Drag** the file into Claude Code chat, or install to `~/.config/claude/skills/`
3. **Start building** — Just describe what you want:
   - *"Create a ReAct agent with web search and calculator tools"*
   - *"Set up RAG with Ollama embeddings and Qdrant"*
   - *"Add Postgres checkpointing for resumable conversations"*

Claude Code will generate working code using Wesichain patterns instantly.

## Requirements

- Rust `1.75+`
- Tokio runtime for async execution
- (Optional) Claude Code with [Wesichain Skills Pack](https://github.com/wesichain/wesichain/releases)

## Minimal Agent/Graph Setup

```toml
[dependencies]
tokio = { version = "1", features = ["macros", "rt-multi-thread"] }
wesichain-core = "0.3.0"
wesichain-llm = "0.3.0"
wesichain-graph = "0.3.0"
```

## Add Retrieval / RAG

```toml
[dependencies]
wesichain-retrieval = "0.3.0"
wesichain-rag = "0.3.0"
```

## Add Checkpoint Persistence

```toml
[dependencies]
wesichain-checkpoint-sqlite = "0.3.0"   # local/embedded
# or
wesichain-checkpoint-postgres = "0.3.0" # production DB
# or
wesichain-checkpoint-redis = "0.3.0"    # low-latency cache/session flows
```

## Anthropic / Claude

```toml
[dependencies]
wesichain-anthropic = "0.3.0"
```

## Coding Agent Stack

```toml
[dependencies]
wesichain-tools = { version = "0.3.0", features = ["fs", "exec", "git"] }
wesichain-agent = "0.3.0"
wesichain-session = "0.3.0"
wesichain-mcp = "0.3.0"   # optional: MCP server connections
```

## HTTP Server

```toml
[dependencies]
wesichain-server = "0.3.0"
```

## Observability

```toml
[dependencies]
wesichain-langsmith = "0.3.0"   # LangSmith
# or
wesichain-langfuse = "0.3.0"    # Langfuse
# or
wesichain-otel = "0.3.0"        # OpenTelemetry
```

## CLI

```bash
cargo install wesichain-cli  # wesichain new / wesichain run
```

## Notes

- The framework is modular-first; avoid installing crates you do not need.
- `ReActGraphBuilder` is the recommended entry point for tool-using ReAct workflows.
- `ReActAgentNode` remains available for compatibility but is no longer the preferred path.
